คู่มือรวม

หลักการใช้งานร่วมกันของทุกเครื่องมือ โดยเฉพาะการบันทึกหลาย Batch การพิมพ์ และการแชร์จากมือถือ

ก่อนเริ่มใช้งาน

  1. เปิดเครื่องมือด้วยเบราว์เซอร์เดิมและอุปกรณ์เดิมเมื่อต้องการกลับมาใช้ข้อมูลต่อ
  2. ตรวจชื่อสูตร หมายเลข Lot และวันที่ก่อนเริ่มจับเวลา
  3. เปิดเสียงของเครื่องและอนุญาตเสียงในเบราว์เซอร์ หากต้องใช้การแจ้งเตือน
  4. เมื่อกรอกอุณหภูมิ น้ำหนัก หรือเวลา ให้ตรวจหน่วยที่แสดงข้างช่องอีกครั้ง
ข้อมูลส่วนใหญ่บันทึกอัตโนมัติในเบราว์เซอร์เครื่องนี้ ไม่ได้ส่งไปเก็บบนเซิร์ฟเวอร์ และอาจหายได้หากล้างข้อมูลเว็บไซต์ เปลี่ยนเบราว์เซอร์ หรือเปลี่ยนเครื่อง

การใช้หลาย Batch

  1. เลือก Batch เดิมจากช่อง “Batch ที่กำลังใช้งาน” เพื่อดูหรือบันทึกต่อ
  2. กด “Batch ใหม่” เมื่อต้องการเปิดงานอีกชุด โดย Batch เดิมจะยังอยู่
  3. ตั้งชื่อสูตรหรือผลิตภัณฑ์ให้ชัดเจน รายการ Batch จะใช้ชื่อนี้ช่วยแยกงาน
  4. กด “ลบ Batch นี้” เฉพาะเมื่อแน่ใจ เพราะข้อมูล Batch นั้นจะถูกลบจากเครื่อง
อย่าใช้ปุ่ม “เริ่มใหม่/ล้างข้อมูล” เมื่อต้องการสร้างงานชุดใหม่ ให้ใช้ “Batch ใหม่” เพื่อไม่ให้ข้อมูลชุดเดิมหาย

พิมพ์และแชร์ PDF

  • พิมพ์เอกสาร เปิดมุมมองพื้นขาวก่อน แล้วสั่งพิมพ์หรือบันทึกเป็น PDF ได้
  • แชร์ PDF สร้างไฟล์เพื่อเปิดเมนูแชร์ของมือถือ หากเครื่องหรือเบราว์เซอร์ไม่รองรับ ระบบจะดาวน์โหลดไฟล์แทน
  • ตรวจหน้าสรุป ชื่อสูตร และ Lot ก่อนพิมพ์ทุกครั้ง

การจับเวลาและเสียงเตือน

ตัวจับเวลาอิงนาฬิกาจริงของเครื่อง จึงยังคำนวณต่อเมื่อสลับแอปหรือดับหน้าจอ อย่างไรก็ตามระบบปฏิบัติการบางรุ่นอาจชะลอเสียงแจ้งเตือนเมื่อเบราว์เซอร์อยู่เบื้องหลัง ควรเปิดหน้าจอไว้ในขั้นตอนสำคัญ
